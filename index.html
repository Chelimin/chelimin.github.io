HOLT

HOLT - a revamped donor app for the 2016 Facebook Hackathon built on Angular and Firebase. To help increase donations and publicity for the Beneficiary, the app includes interactive elements, an Admin dashboard, as well as Facebook login and graph API. The project won "Best All-Rounded Product" in the 12 hour competition.

Twitterlyzer

Twitterlyzer - an app for analyzing Tweet sentiments for the requested keywords built on Node.js, using the IBM Watson Alchemy and Twitter APIs. This can be used to analyze market sentiments on People, Companies, Products, Services, or Campaigns.

DevHub

DevHub - a "Linkedin" for Developers built on Node.js. Using Github's API, raw data is converted into meaningful metrics to display data visualisation on the client-side. This is to show the Developer's expertise and activity, as well as to facilitate finding Developers for a particular language/framework.

Forces

Forces - an Apple iBook published on iTunes and built with iBook Author. It contains animations, videos, and interactive questions for hands-on learning.

HouseHelp

HouseHelp - a fully interactive, mobile-responsive website built on HTML, JavaScript and Bootstrap; with Google Maps API and form input.

Cat's List

Cat's List - a Craig's list replica for housing services; CRUD web app built on Ruby on Rails with integrated Facebook API.
